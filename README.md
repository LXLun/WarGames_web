# ⚔️ 网页兵棋推演工具 (Web Wargame Tabletop)

这是一个基于 Web 的轻量级兵棋推演平台，专为桌面兵棋爱好者设计。它提供了一个虚拟的地图桌面，允许用户上传地图、算子，并进行回合制的推演操作。无需安装复杂的软件，打开浏览器即可开始推演。

## ✨ 主要功能

### 🗺️ 地图与场景
- **自定义地图**：支持上传本地图片作为推演底图。
- **持久化存储**：利用浏览器数据库（IndexedDB）自动保存当前的棋局状态，刷新页面或关闭浏览器后数据不丢失。
- **初设模式 (Setup Mode)**：提供专门的初始设置阶段，配置完成后可保存为“初始局面”，游戏中随时可以一键“重置到初设”。

### 🧩 算子系统 (Tokens)
- **多层级算子库**：
  - **主算子**：核心作战单位，支持分别绑定正面和背面图像（双面算子）。
  - **子算子**：可独立使用，也可绑定到主算子逻辑中（如拆分规则）。
  - **状态标记 (Markers)**：用于标记混乱、溃退、工事等状态，支持无限制拖拽使用和大小缩放。
  - **图表 (Tables)**：可将规则表、CRT（战斗结果表）贴在地图旁随时查看。
- **丰富的交互**：
  - **拖拽与移动历史**：移动算子时自动显示**移动轨迹**（红色虚线箭头）和近似移动距离，方便复盘和确认步数。
  - **堆叠系统 (Stacking)**：自动识别地理位置重叠的算子。拖动堆叠最底部的算子（Leader），整个堆叠会跟随移动；按住 `Shift` 键拖动可分离单个算子。
  - **算子状态**：支持翻转（Flip）、激活/未激活状态切换、层级调整（置顶）。
  - **右键菜单**：提供翻面、标记、删除、克隆、放入区域等快捷操作。

### 🎲 辅助工具
- **3D 骰子**：内置物理引擎的 3D 骰子投掷模拟，支持多种骰子类型。
- **区域管理 (Zones)**：
  - **阵亡区 (Graveyard)**：用于收纳被消灭的单位，支持拖拽复活回地图（复活时状态重置）。
  - **抽签袋 (Draw Bag)**：模拟盲抽机制，支持将算子放入袋中并随机抽取。
- **回合控制**：
  - 记录当前回合数。
  - **结束回合**：一键清除所有算子的移动轨迹（红线）和行动状态，准备下一回合。

### 💾 导入与导出
- **JSON 存档**：支持将当前的整个棋局配置（包括所有算子库图片、地图、当前布局）导出为 `.json` 文件。
- **分享与备份**：生成的配置包可以发送给朋友，或作为特定剧本的存档随时导入恢复。

## 🛠️ 技术栈

- **核心框架**: [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- **构建工具**: [Vite](https://vitejs.dev/)
- **图形渲染**: [Konva](https://konvajs.org/) / [react-konva](https://github.com/konvajs/react-konva) (高性能 Canvas 交互)
- **状态管理**: [Zustand](https://github.com/pmndrs/zustand)
- **样式方案**: [Tailwind CSS](https://tailwindcss.com/)
- **本地存储**: [localforage](https://github.com/localForage/localForage)
- **动画效果**: [Framer Motion](https://www.framer.com/motion/)

## 🚀 快速开始

### 1. 环境准备
确保本地已安装 Node.js (推荐 v18+)。

### 2. 安装依赖

```bash
npm install
# 或者
yarn install
```

### 3. 启动开发服务器

```bash
npm run dev
```

浏览器访问 `http://localhost:5173` 即可看到界面。

### 4. 构建生产版本

```bash
npm run build
```

构建产物将输出到 `dist` 目录。

## 📖 使用指南

1.  **准备素材**：准备好你的地图图片和算子切片图片。
2.  **上传地图**：在左侧栏点击“上传地图”。
3.  **上传算子**：在左侧栏“算子库”区域上传算子图片。
    - 点击算子右下角的 `+` 可绑定背面图片。
4.  **开始推演**：
    - 将算子从侧边栏拖拽到地图上。
    - 此时默认处于 **"初设模式"**。摆放好初始队形后，点击 **"保存当前为初设"**。
    - 点击 **"开始游戏"** 进入第一回合。
5.  **游戏中**：
    - 移动算子进行机动，红线会显示移动路径。
    - 右键算子进行战斗结算前的状态标记。
    - 阶段结束时，点击左下角的 **"结束本回合"** 清理盘面状态，进入下一回合。
